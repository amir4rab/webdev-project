## توجه داشته باشید این آموزش فقط برای سیسیتم هایه تحت دبین قابل اجرا می باشد

برسیه آپدیت هایه موجود
```bash
sudo apt update
```

اعمال آپدیت ها
```bash
sudo apt upgrade
```
